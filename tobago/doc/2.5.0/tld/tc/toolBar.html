Tag toolBar

Renders a toolbar.

Allowed subcomponents are subtypes of UICommand i.e. 'button' and 'link' tags. These are rendered by ToolbarRenderer, so the result has no difference.

To add an drop down menu to a button add a facet 'dropDownMenu' containing a tag to the button. Label's and Image's on those menu tag's are ignored and replaced by the renderer.
